# Compliance Boundary

本项目只用于个人求职分析和学习研究，不能扩展成自动化骚扰、批量采集或公开分发原始平台数据的工具。

## 必须遵守

- 低频采集，不做大规模抓取。
- 不自动投递简历。
- 不自动发送消息。
- 不采集与岗位无关的个人隐私。
- 不提交 Cookie、Chrome Profile、`.env`、个人简历原文或真实原始岗位数据。
- 默认使用专用 Chrome Profile，不复用主浏览器敏感登录态。
- 尊重平台规则和网站使用条款。

## 公开仓库策略

公开版本只保留：

- 脱敏样例数据。
- 数据结构和处理代码。
- 评估脚本和可复现实验说明。
- 不包含真实账号状态、密钥、Cookie、个人简历或平台原始数据。
